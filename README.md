# BrainBeat
 An app that helps students track their study sessions with personalized analytics like a “Spotify Wrapped” for learning

 Throughout the semester, students can use BrainBeat to see how long and how many time they were locked in every day. The app will log the study sessions. At the end of the smester, the students get a personalized showcase of their study habits and streaks and total focus time.

 # Features
 Timing study session with when a student began and when they ended
 Keeping track of the number of study sessions and if the student has streaks
 "Spotify wrapped" style of stats summary for the student

 # Stack
 Backend is with Python
 Frontend is with React Native
 Database to store stuff is with Firebase
